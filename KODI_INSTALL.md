# Sabri Maranan Subtitles - Kodi Installation Guide

## ✅ CORRECT INSTALLATION METHOD

### Step 1: Download the Repository Addon ZIP
Download this file to your device:
```
https://raw.githubusercontent.com/motekamazon-droid/repo/main/zips/repository.sabri-maranan/repository.sabri-maranan-1.0.0.zip
```

Or use shortened URL:
```
bit.ly/sabri-repo-zip
```

### Step 2: Install Repository Addon in Kodi
1. In Kodi: **Settings** → **Add-ons** → **Install from ZIP file**
2. Navigate to and select: `repository.sabri-maranan-1.0.0.zip`
3. Kodi will extract and install the repository addon

### Step 3: Install Sabri Maranan Subtitles from Repository
1. Go to: **Settings** → **Add-ons** → **Install from repository**
2. Select: **Sabri Maranan Repository**
3. Browse to: **Services**
4. Install: **Sabri Maranan Subtitles**

### Step 4: Enable the Service
1. Go to: **Settings** → **Add-ons** → **Services** → **Sabri Maranan Subtitles**
2. Click **Enable**

### Step 5: Use It
1. Play any Sabri Maranan episode in Kodi
2. Subtitles will automatically load within 5 seconds
3. No configuration needed!

## Troubleshooting

### "Can't find the repository addon ZIP"
- Make sure you downloaded the correct ZIP file from the raw GitHub URL
- Don't use the HTML page URL, use the raw.githubusercontent.com URL

### "No subtitles loading"
1. Check that the addon is **enabled** in Services settings
2. Check Kodi logs: Settings → System → Logging → Enable debug logging
3. Look for messages from "Sabri Maranan Subtitles" service
4. Check internet connection

### "Addon installed but not in Services"
- Make sure to install the **Sabri Maranan Subtitles** addon (not the repository)
- Repository addon is for browsing, the Subtitles addon is what you need enabled

### "Episode not found"
- Subtitle may not be available yet
- Check if episode number is 1-17 (others being processed)
- File must have episode number in filename for addon to detect it
